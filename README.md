# Lillith Aero

A simple indie 2D platformer game prototype developed in the Godot Engine

## About the Project

Lillith Aero tells the story of Lillith, a woman that gets stuck in her computer and has to find a way out. The game is heavily inspired by Frutiger Aero and other dominant styles of the 2000s internet era.

## Technologies Used

- **Programming Language:** C#  
- **Game Engine:** Godot Engine

## Notes

- This Project was made to test and upgrade my programming skills and to understand Godot deeply
- This is an early-stage prototype. Development is ongoing and subject to change.
